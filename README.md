# NegyedikBeadando

A célom egy olyan torpedó játék létrehozása volt, amelyben lehetőség van multiplayer játékmódra szerver segítségével.
A játék "grafikája" javafx segítségével készült. (tudom hogy picit elavult, de ez volt a legegyszerűbb)

A játék indítása:
1) negyedikBeadandoSzerver package futtatása
2) negyedikBeadandoPcg package futtatása, felugró játékparaméterező ablakon a kívánt opciók beállítása, és az első játékos számára a játék indítása.
3) negyedikBeadandoPcg package futtatása, felugró játékparaméterező ablakon a ugyanazon opciók beállítása, és a második játékos számára a játék indítása.

(Az első tábla  a saját tábla, a másodikon lehet találgatni)
